# Arrays in Java

## Content

Arrays are a fundamental data structure in Java. They are used to store multiple values of the same type in a single variable.

## Code Examples

```
int[] numbers = new int[5];
for(int i = 0; i < numbers.length; i++) {
  numbers[i] = i;
}
```
